# MONTIO INC Single-Page Website

Dark, modern, responsive nonprofit research website for **MONTIO INC** (`montio.org`), built as a single static HTML page.

## Run locally

1. Open a terminal in this project:
   ```bash
   cd /workspace
   ```
2. Serve the files:
   ```bash
   python3 -m http.server 8000
   ```
3. Open:
   `http://localhost:8000`

## Deploy

Deploy `index.html` to any static hosting platform:

- GitHub Pages
- Netlify
- Vercel (static)
- AWS S3 + CloudFront

No build step is required.
